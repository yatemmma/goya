goya
====

Hello from the sea!

## Usage

install [cliclick](https://github.com/BlueM/cliclick)

```
$ brew install cliclick
```

start server

```
$ bundle install
$ bundle exec rake
```

and open page on browser

```
$ open app/html/index.html
```

enjoy cruising!

## Support

* MacOS X Yosemite
* Chrome
* ruby 2.1.2
